# AutoIT Projesi

Merhaba! Bu bir AutoIT projesidir.

## Proje Yapısı

```
.
├── src/
│   └── main.au3          # Ana script dosyası
├── .vscode/
│   └── launch.json       # VS Code çalıştırma ayarları
└── README.md             # Proje belgesi
```

## Başlangıç

1. `src/main.au3` dosyasını açın
2. F5 tuşuna basın veya Debug menüsünden "Run" seçeneğini tıklayın
3. Script çalışacaktır!
